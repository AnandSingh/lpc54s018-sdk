Aruba_Flashless_MCM USB Audio DMIC Example
==========================

Example description
-------------------
The example shows how to use USBD ROM stack to create and USB AUDIO class
device. This device supports 2 channels audio in.
Default supported format is 16000 Hz sampling rate with 16-bit sample width. For recording audio open the voice recording application (example in windows 10 user Voice Recorder App), to record audio from speaking to DMIC.

Special connection requirements
-------------------------------
For downloading code and debugging connect an USB cable to micro
connector (J8) and host.
Connect another USB cable between micro connector (J3) on board
and to a host.
